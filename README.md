# HTTP GET Method Practice

## Overview
This project demonstrates how to send an HTTP GET request using JavaScript's Fetch API. It fetches user data from a public REST API and displays the response status along with the complete JSON response.

## Features
- Send an HTTP GET request
- Fetch data from a public REST API
- Display request status code
- Show the JSON response
- Loading indicator while the request is in progress
- Simple and responsive user interface

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- Bootstrap 4
- Fetch API

## How to Run
1. Download or clone the project.
2. Open `index.html` in a web browser.
3. Click **Send Get Request**.
4. View the request status and API response.
